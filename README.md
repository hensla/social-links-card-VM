Google Fonts — Inter

✨ Funcionalidades

Card centralizado com foto de perfil em formato circular
Nome, localização e descrição do usuário
Lista de links para redes sociais (GitHub, Dribbble, LinkedIn, Instagram, X)
Efeito hover na foto com escala e mudança de cor da borda
Design responsivo
Tema escuro com paleta de cinzas
